# 침입 탐지 시스템 PRD

## 프로젝트 개요
본 프로젝트는 랜덤 포레스트와 강화학습을 결합한 하이브리드 접근 방식을 통해 네트워크 패킷을 실시간으로 분석하고 침입을 탐지하며 자동으로 대응하는 시스템을 구현한다.

## 핵심 기능 요구사항

1. 실시간 패킷 캡처
   - 네트워크 인터페이스에서 패킷을 실시간으로 캡처
   - 캡처된 패킷 데이터 분석 및 전처리
   - Scapy 라이브러리를 활용한 패킷 처리

2. 랜덤 포레스트 기반 침입 탐지
   - 패킷 데이터를 분석하여 비정상 트래픽 패턴 탐지
   - 데이터 전처리 및 특성 추출
   - 모델 학습 및 저장
   - 메모리 사용량 최적화를 위한 청크 단위 처리

3. 강화학습 기반 대응 모델
   - DQN(Deep Q-Network) 기반 에이전트 구현
   - 랜덤 포레스트 모델의 예측 결과를 환경 상태로 활용
   - 최적의 대응 방식 학습 및 적용

4. 이중 운영 모드
   - 경량 모드: 기본 특성 7개 사용, 모든 환경 지원
   - 고성능 모드: 확장 특성 12개 사용, 수리카타 엔진 통합

5. GUI 및 모니터링
   - 시스템 상태 및 탐지된 위협에 대한 시각화
   - 머신러닝 모델 성능 모니터링
   - 메모리 사용량 최적화 필요

6. 메모리 최적화
   - 대용량 데이터 처리를 위한 메모리 사용량 최적화
   - GUI 컴포넌트의 자원 사용량 감소
   - 장기 실행 안정성 향상

## 기술 스택
- Python
- Scikit-learn (랜덤 포레스트)
- PyTorch (강화학습)
- Scapy (패킷 캡처)
- PyQt6 (GUI)
